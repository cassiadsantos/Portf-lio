## 🤖 Aprendizados em Inteligência Artificial

Durante o curso, explorei profundamente o universo da IA:

### IA para Construção de Portfólio
- **Design com IA**: Geração de ideias visuais, cores e layouts
- **Conteúdo com IA**: Criação de descrições profissionais e atraentes
- **Código com IA**: Otimização e refatoração inteligente
- **Imagens**: Geração e edição de assets visuais
- **SEO**: Melhor visibilidade nos buscadores

### Aplicações de IA em Diferentes Escopos
- 💼 **Negócios**: Automação, análise de dados, previsões
- 🏥 **Saúde**: Diagnóstico, pesquisa, personalização
- 🎓 **Educação**: Tutoria personalizada, análise de progresso
- 🎮 **Entretenimento**: Recomendações, geração de conteúdo
- 🚗 **Mobilidade**: Carros autônomos, otimização de rotas
- 💻 **Desenvolvimento**: GitHub Copilot, completação de código

### Dicas de Prompt Engineering
1. **Seja Específico**: Detalhe exatamente o que quer
2. **Contexto é Chave**: Forneça informações relevantes
3. **Estruture a Requisição**: Use listas e separadores
4. **Iteração Progressiva**: Refine gradualmente o resultado
5. **Exemplos e Referências**: Use padrões conhecidos
6. **Defina o Resultado Esperado**: Seja claro no formato

### Ferramentas de IA Exploradas
- **ChatGPT**: Conversa e geração de texto
- **DALL-E / Midjourney**: Geração de imagens
- **GitHub Copilot**: Assistente de código
- **Claude**: Análise e escrita profissional
- **Perplexity**: Busca com IA
- **Figma AI**: Design assistido por IA

### HTML Semântico
- `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`
- Meta tags para viewport e responsividade
- Estrutura organizada e acessível

### CSS Avançado
- Variáveis CSS para temas
- Flexbox e CSS Grid
- Media queries para responsividade
- Animações e transições
- Gradientes lineares

### JavaScript Moderno
- Seleção de elementos com querySelector
- Event listeners e delegação
- localStorage para persistência
- Intersection Observer para animações ao scroll
- Template literals

## 🌐 Publicação no GitHub Pages

1. Crie um repositório no GitHub
2. Faça commit dos arquivos:
   ```bash
   git add .
   git commit -m "Initial commit: Jornada de Aprendizado"
   git push origin main
   ```

3. Vá para **Settings** > **Pages**
4. Selecione **Deploy from a branch**
5. Escolha **main** como branch
6. Salve e aguarde o deploy

## 📝 Customização

### Alterar Nome
- Edite "Estudante" em `index.html`

### Alterar Cores
- Modifique as variáveis CSS em `style.css`:
  ```css
  --primary-color: #6366f1;
  --secondary-color: #ec4899;
  ```

### Adicionar Projetos
- Duplique a estrutura `.project-card` em `index.html`
